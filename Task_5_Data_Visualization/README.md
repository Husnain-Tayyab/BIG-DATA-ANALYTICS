# Task 5: Data Visualization Portfolio

## Overview
This final task focuses on data storytelling. Using a generated time-series dataset of weather patterns across multiple cities, it utilizes Matplotlib and Seaborn to create a comprehensive visualization portfolio featuring over 10 different chart types.

## Key Visualizations Created
* Line plots for time-series analysis.
* Bar charts and histograms for data distribution.
* Scatter plots for correlation analysis.
* Box plots and violin plots for statistical density.
* Heatmaps for displaying correlation matrices.
* Multi-panel figures (subplots) with customized colors, labels, and legends.

* Jupyter Notebook (`.ipynb`) with complete source code.

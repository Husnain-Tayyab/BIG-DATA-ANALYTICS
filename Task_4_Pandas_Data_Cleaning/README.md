# Task 4: Pandas Data Exploration and Cleaning

## Overview
Using the widely recognized Titanic dataset, this task demonstrates real-world data wrangling and cleaning techniques. It transforms raw, imperfect data into a structured format ready for machine learning algorithms.

## Key Skills Demonstrated
* Loading data dynamically from CSV, JSON, and Excel formats.
* Initial data exploration (`head`, `info`, `describe`, `shape`).
* Filtering data using complex boolean indexing.
* Sorting datasets by multiple columns.
* Handling missing data (`isnull`, `dropna`, `fillna` using medians).
* Feature engineering (adding, renaming, and dropping columns).
